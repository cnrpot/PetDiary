# 宠物日记 (Pet Diary) 🐾

一款基于现代 Android 技术栈开发的治愈系宠物成长记录应用。

## 🌟 项目简介

“宠物日记”是一款专为宠物主设计的安卓应用。它采用了清新、温馨的 UI 风格，旨在通过简单的操作，记录宠物生命中每一个珍贵的瞬间。

## ✨ 核心功能

*   **多宠管理**：支持为多只宠物建立档案，记录品种、性别、体重及专属照片。
*   **成长日记**：瀑布流式的时间轴日记，支持添加标题、天气、心情图标和精美排版。
*   **健康提醒**：内置喂食、洗澡、疫苗、驱虫等多种提醒类型，守护宠物健康。
*   **体重追踪**：数字化记录宠物体重变化，随时掌握成长动态。
*   **自适应布局**：完美适配手机、折叠屏及平板设备。

## 🛠️ 技术栈

*   **UI 框架**：Jetpack Compose (声明式 UI)
*   **架构模式**：MVVM (Model-View-ViewModel)
*   **持久化层**：Room Database (本地 SQLite 存储)
*   **导航**：Jetpack Navigation 3 (State-driven)
*   **图片加载**：Coil
*   **主题设计**：Material Design 3 (可爱治愈系定制主题)
*   **核心语言**：Kotlin

## 🚀 运行指南

1.  克隆仓库：`git clone https://github.com/cnrpot/PetDiary.git`
2.  使用 **Android Studio Ladybug (2024.2.1)** 或更高版本打开项目。
3.  等待 Gradle 同步完成（建议配置国内镜像源以加速）。
4.  点击 **Run** 按钮，在 Android 15 (API 35+) 模拟器或真机上运行。
